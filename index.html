Gym Database & Management Software:

This project is still in progress and is a dynamic rendering web application for Gym Management. You can view the source code of the project on my GitHub.

Gym Owner Features
  • Login/Authentication: Gym Owners log in using their username and password. On successful authentication, they are redirected to their dashboard.
  • Gym Owner Dashboard:
    • Gym Count: Total number of gyms owned.
    • Gym Data: List of gyms owned with member count.
    • Gym Details: Displays gym name and member count.
  • Gym Ownership: Gym Owners can own multiple gyms (Many-to-Many relationship).
  • Gym Check-In Management: Manage gym check-ins and ensure only authorised users can access data.
  • Access Control: Gym Owners can only view and manage data for gyms they own.
Member Features
  • Login/Authentication: Members log in using their username and password to access their dashboard.
  • Member Dashboard:
    • Gym Name: The name of the gym the member is associated with.
    • Join Date: The date the member joined the gym.
    • Gym Code: A unique code assigned for check-in/check-out.
    • Member Name: The member's full name or username.
  • Gym Code Generation: Unique codes are generated upon saving member profiles.
  • Gym Check-In: Members use gym codes to check in/out, creating visit records.
  • Visit Management: Tracks member entry and exit times for the gym.
Gym Check-In Functionality
  • Access Control: Only Gym Owners can access the check-in page.
  • Check-In Process:
    • Members check in with their gym code, creating a new visit record with the current entry time.
    • Members check out by updating their visit's exit time.
    • Ensures valid gym codes and associations.
  • Error Handling:
    • Invalid codes or unauthorised access result in error messages.
    • Prevents members from checking out without an active visit.
